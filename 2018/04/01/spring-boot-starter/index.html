Springboot quick start

Spring boot can let you create a stand ablone spring project, which means you don’t need any servlet container.

Create Springboot project by using maven

Create maven project, and don’t use any archetype. Then add this dependencies in pom.xml.

Create class and a Main Method

In spring boot project, we need a main method to run this application just like any java application. I think it’s amazing because I never use main function to run a web server.

Noticed that we use @SpringBootApplication annotation to let spring boot know it is the main method. In method use SpringApplication.run to run this application. The first parameter is the main class, and the second parameter is the parameter passed into this main function. It does these things for us:

  1. Sets up default configuration
  2. Starts Spring application context
  3. Performs class path scan
  4. Starts Tomcat server
