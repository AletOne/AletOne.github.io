Spring boot can let you create a stand ablone spring project, which means you don’t need any servlet container.

Create Springboot project by using maven

Create maven project, and don’t use any archetype. Then add this dependencies in pom.xml.

Create class and a Main Method

In spring boot project, we need a main method to run this application just like any java application. I think it’s amazing because I never use main function to run a web server.

Noticed that we use @SpringBootApplication annotation to let spring boot know it is the main method. In method use SpringApplication.run to run this application. The first parameter is the main class, and the second parameter is the parameter passed into this main function. It does these things for us:

  1. Sets up default configuration
  2. Starts Spring application context
  3. Performs class path scan
  4. Starts Tomcat server

Start to Learn REST API

Recently, my resume is always rejected by companies. And I found they all like candidates who have knowledge about REST. Okay……Nothing hard.

Protocol

format of message.

Rest uses XML/JSON as message format.

Restful web service doesn’t need documentation.

Rest and HTTP

Http method:

  • Get : read-only, repeatable aka idempotent
  • Post ： cannot be repeatable
  • Put ： repeatable
  • Delete : repeatable

Http response code

  • 1xx informational
  • 2xx Success: 200 ok; 201 Created; 202 No Content;
  • 3xx Redirection : 302 Found; 304 Not Modified; 307 Temporary Redirect;
  • 4xx Client Error: 400 Bad Request; 401 Unauthorized; 403 Forbidden; 404 Not Found; 405 Unsupported Media Type;
  • 5xx Server Error: 500 Internal Server Error; 50

Level 0, 1, 2, 3 in RESTful

  • level 0: One URI, everything in xml.
  • leve 1: Resource URI. Individual URIs for each resource.
  • leve 2: Use HTTP methods.
  • level 3: HATEOAS. Responses have linkes that the clients can use.

Start With JAX RS

  1. create a project by maven : use glassfish-jersey-quickstartwebapp as a templet.
  2. start tomcat.
  3. add your own resources class.
  4. add @Path(“/resources name”) annotation in resources class.
  5. add @GET to specific java method.
  6. Annotate method with @Produces to specifying response format.

some notation

  • @XmlRootElement: Tell Jersey list of this class can be convert to xml.
  • @Consumes(mediatype): post parameter type
  • @QueryParam: /aaaa?year=…&asf=advdv
  • @MatrixParam(“some param name”): /aaaa;param=value
  • @HeaderParam(“custom”): cumtom header value
  • @CookieParam(“cookie name”): get cookie
  • @FormParam(): acces form submissions
  • @Context: get context.
  • @BeanParam
  • @XmlTransient
  • @Provider: register in Jax-rs as a provider
